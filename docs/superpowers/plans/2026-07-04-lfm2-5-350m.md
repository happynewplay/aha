# LFM2.5-350M 实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 为 `aha` 新增 `LFM2.5-350M` safetensors 推理支持，接入 CLI / API / registry，并将模型原生 `<|tool_call_start|>...<|tool_call_end|>` 输出适配为标准 OpenAI `tool_calls`。

**架构：** 新增独立 `src/models/lfm2_5` family，按 `config.layer_types` 构建 `Conv` 与 `FullAttention` 混合文本模型。运行时只实现 safetensors 初始化；`gguf` / `onnx` 在 `artifact` 与 loader 路由层声明支持，但在 family 初始化时返回明确的未实现错误。普通 completion 和 stream completion 都走标准 OpenAI response builder，tool-call 由生成器和 `utils` 层共同完成标签解析与标准字段组装。

**技术栈：** Rust 2024、candle-core / candle-nn / candle-transformers、rocket、clap、aha_openai_dive、minijinja、serde / serde_json

在 `src/models/lfm2_5/generate.rs` 实现流式状态：

```rust
#[derive(Default)]
struct Lfm2_5StreamState {
    tool_call_id: Option<String>,
    tool_call_content: String,
}

impl Lfm2_5StreamState {
    fn push(&mut self, decoded_token: &str, model_name: &str) -> Result<Option<ChatCompletionChunkResponse>> {
        match decoded_token {
            "<|tool_call_start|>" => {
                self.tool_call_id = Some(uuid::Uuid::new_v4().to_string());
                self.tool_call_content.clear();
                Ok(None)
            }
            "<|tool_call_end|>" => {
                let chunk = build_completion_chunk_response(
                    decoded_token.to_string(),
                    model_name,
                    self.tool_call_id.clone(),
                    Some(self.tool_call_content.clone()),
                );
                self.tool_call_id = None;
                self.tool_call_content.clear();
                Ok(Some(chunk))
            }
            _ if self.tool_call_id.is_some() => {
                self.tool_call_content.push_str(decoded_token);
                Ok(None)
            }
            _ => Ok(Some(build_completion_chunk_response(
                decoded_token.to_string(),
                model_name,
                None,
                None,
            ))),
        }
    }
}
```

        let mut model = Lfm2_5GenerateModel::init_from_spec(spec, None, None)?;
        let result = model.generate(Self::build_text_request(input)?)?;
        let result_json = serde_json::to_string_pretty(&result)?;
        println!("Result: {}", result_json);
        if let Some(out) = output { std::fs::write(out, &result_json)?; }
